# Project Roadmap

## Sprint 1: Foundation

- Create repository structure.
- Define business problem.
- Define data dictionary.
- Create SQL schema.
- Add synthetic sample documents.
- Build first parser.
- Export clean CSV.

## Sprint 2: Database and analytics layer

- Load extracted records into PostgreSQL.
- Create staging and analytical tables.
- Add SQL queries for KPIs.
- Prepare Power BI connection.

## Sprint 3: Power BI dashboard

- Build Executive Overview.
- Build Process Performance page.
- Build Data Quality page.
- Build Operational Insights page.
- Add screenshots to repository.

## Sprint 4: Professionalization

- Add dbt models.
- Add automated tests.
- Add GitHub Actions.
- Improve error handling and logging.

## Sprint 5: Cloud-ready version

- Move documents to Azure Blob Storage.
- Integrate Azure Document Intelligence.
- Deploy PostgreSQL/Azure SQL target.
- Add Airflow orchestration.
